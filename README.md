Variables

Example:

Data Types

- Float

- List

---

Conditional Statements

Conditional statements allow programs to make decisions.

Example:

if age >= 18:
    print("Adult")
else:
    print("Minor")

---

Loops

Loops repeat actions.

Example:

for i in range(5):
    print(i)

---

Functions

Functions help organize code into reusable blocks.

Example:

def greet():
    print("Hello")

---

Error Handling

Error handling prevents programs from crashing unexpectedly.

Example:

try:
    number = int(input())
except ValueError:
    print("Invalid input")

---

File Handling

Python can read and write files.

Example:

with open("notes.txt", "r") as file:
    print(file.read())
